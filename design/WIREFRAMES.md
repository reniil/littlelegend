# LittleLegend Wireframes & User Flows

## 🎯 User Flows

### Flow 1: Parent Signup → First Entry
Landing Page → Signup → Create Child Profile → Add Activity → Dashboard

### Flow 2: Adding New Activity
Dashboard → Click "Add" → Select Category → Fill Form → Upload Evidence → Save

### Flow 3: Viewing Timeline
Dashboard → Timeline Tab → Scroll Years → Click Activity → View Details

### Flow 4: Generating CV
Dashboard → CV Tab → Select Age → Preview → Download PDF

---

## 📱 Screen Wireframes

### 1. Landing Page
Hero section, features grid, testimonials, pricing, CTA

### 2. Signup/Login Page
Email/Google/Apple signup options

### 3. Onboarding
3-step: Create profile, choose avatar, welcome tour

### 4. Dashboard
Stats cards, recent activity, category breakdown, timeline tabs

### 5. Add Activity Modal
Category selection, form fields, evidence upload

### 6. Timeline View
Chronological journey, grouped by year

### 7. CV Preview
Professional CV formatted from childhood data
